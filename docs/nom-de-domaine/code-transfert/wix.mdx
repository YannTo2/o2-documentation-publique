# Récupérer le code de transfert chez WIX

Ce guide explique comment **débloquer** votre nom de domaine et récupérer le **code de transfert** (authcode) lorsque
votre nom de domaine est réservé chez WIX. 

Le fait de débloquer le nom de domaine et obtenir l'authcode permet de transférer le nom de domaine chez un autre
registre de domaine, par exemple [votre hébergement web o2switch](https://www.o2switch.fr).

Si vous ne savez pas comment fonctionne un transfert de nom de domaine, c'est-à-dire comment transférer la gestion du
domaine d'un registre à un autre, on explique cela plus en détails sur la page [Comment transférer un nom de domaine
?](/nom-de-domaine/transferer-un-domaine).

## Procédure pour récupérer le code de transfer chez WIX

Dans un premier temps, connectez-vous sur l'interface WIX via [ce lien](https://users.wix.com/signin?forceRender=true).

<Badge text="1" />Une fois connecté, la page d'accueil s'affiche. Depuis cette page, il faut cliquer sur `Web Cloud`
situé dans le menu supérieur. 

Suite à cela, vous êtes redirigé vers une page différente qui présente les outils en rapport avec les services web de
WIX.

<Badge text="2" /> Ensuite, il faut cliquer sur `Domaines` afin d'accéder à la page de gestion de vos domaines chez Wix

<Badge text="3" /> Puis cliquer sur les ... à droite du domaine que vous souhaitez transférer.

<Image
    src="/img/docs/nom-de-domaine/code-transfert/recuperer-code-transfert-wix.png"
    title="Première étape pour débloquer le domaine chez WIX"
    caption="Première étape de la procédure pour récupérer le code de transfert chez WIX"
    alt="Capture d'écran de l'interface de gestion WIX pour récupérer le code de transfert"
/>

<Badge text="4" /> Ensuite, il faut désactiver la `Transférer le domaine`. Un nouvel écran s'affiche et vous
demande la confirmation, il faut valider pour continuer.

<Image
    src="/img/docs/nom-de-domaine/code-transfert/confirmation-transfert-wix.png"
    title="Récupération du code de transfert chez WIX"
    caption="Récupération du code de transfert (authcode) chez WIX"
    alt="Capture d'écran de l'interface de gestion WIX pour récupérer le code de transfert"
/>

<Badge text="5" />Vous recevrez ensuite sur votre adresse mail configuré chez Wix le code de transfert de votre domaine.

Vous pouvez ensuite lancer le transfert du domaine depuis votre espace client. Il faut aller dans la rubrique `Gérer mes
services` puis cliquer sur `Détails` en face de la prestation concernée. Dans la page `Détails`, il faut cliquer sur
`Initier le transfert du domaine`. 

Vous avez également la possibilité de transmettre le code de transfert au support, qui se chargera d’initier la procédure.
