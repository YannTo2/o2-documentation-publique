# Récupérer le code de transfert d'un domaine Ionos / 1&1

Dans ce guide, nous allons voir comment récupérer le code de transfert d'un nom de domaine réservé chez 1&1. 

Le fait de
récupérer le code de transfert permet de transférer la gestion totale du nom domaine chez un autre registre, par exemple
vers **[o2switch](https://www.o2switch.fr/hebergement-illimite/)**.

Si vous ne savez pas comment fonctionne un transfert de nom de domaine, c'est-à-dire comment transférer la gestion du
domaine d'un registre à un autre, on explique cela plus en détails sur la page [Comment transférer un nom de domaine
?](/nom-de-domaine/transferer-un-domaine).

:::danger[Récupérez toutes vos données chez Ionos avant de récupérer le code de transfert]
Nous vous recommandons **fortement** de récupérer l'ensemble de vos données hébergés chez Ionos **avant** de lancer
l'étape de récupération du code de transfert. 


En effet, 1&1 à tendance à couper la prestation d'hébergement dés que le transfert du domaine est terminé. Cela ne 
semble pas être systématique mais plusieurs anciens clients Ionos ont déjà été confrontés à ce problème. 

On suppose (sans certitude) que cela arrive sur une prestation d'hébergement qui est liée à un seul domaine chez Ionos.
Le tranfert du domaine doit déclencher une procédure de suppression d'hébergement.

Dans le doute, pensez à récupérer toutes vos données avant de faire cette opération.
:::

Dans un premier temps, connectez-vous sur votre interface client 1&1. Vous pouvez faire cela sur [cette
page](https://login.ionos.fr/). 

<Badge text="1" /> Une fois connecté, vous devez arriver sur la page visible ci-dessous. Sur cette page, cliquez sur
`Domaines & SSL`. Si la page affichée est différente, il faut alors cliquer sur le bouton ``Menu`` situé en haut à gauche.

<Image
    src="/img/docs/nom-de-domaine/code-transfert/1and1-code-tranfert-authcode-1.png"
    title="Se rendre dans la catégorie Domaines & SSL dans l'interface de gestion Ionos"
    caption="Se rendre dans la catégorie « Domaines & SSL »"
    alt="Capture d'écran de la page d'accueil de Ionos"
/>

<Badge text="2" />  Ensuite, une page s'ouvrira et vous fournira une liste de l'ensemble de vos domaines. Il vous suffit 
alors de cliquer sur le bouton d'action ``...`` au niveau du domaine à transférer, puis sur ``Transfert et renouvellement``.

<Image
    src="/img/docs/nom-de-domaine/code-transfert/recuperer-authcode-1and1-2.png"
    title="Page de gestion des domaines du dashboard ionos"
    caption="Sélection de l'outil de gestion du domaine"
    alt="Capture d'écran de l'espace d'administration ionos"
/>

<Badge text="3" /> Sur l'écran suivant, vous devez décocher la case du ``Verrouillage du nom de domaine``. La page va s'actualiser 
et le statut va passer en ``En cours d’actualisation`` durant quelques minutes. Vous pouvez l'ignorer et cliquer sur 
``Afficher le code d'autorisation``.

<Image
    src="/img/docs/nom-de-domaine/code-transfert/recuprer-code-transfert-1and1-3.png"
    title="Débloquer et récupérer le code de transfert d'un domaine chez Ionos"
    caption="Débloquer et afficher le code de transfert chez Ionos"
    alt="Capture d'écran de la procédure de récupération du code de transfert ionos"
/>

<Badge text="4" /> Le code de transfert s'affichera en cliquant sur ``Afficher le code d'autorisation``. Le authcode est sensible à
la casse, les majuscules/minuscules et caractères spéciaux sont importants.

Si la case de verrouillage du domaine, précédemment décochée, est à nouveau cochée cela signifie que le transfert n'est pas encore
possible. Cela peut se produire lorsque le domaine a été acquis / renouvelé récemment.

En attendant de pouvoir lancer le transfert chez o2switch, vous pouvez choisir de [configurer nos serveurs DNS](/nom-de-domaine/changer-dns/ionos-1and1).

<Image
    src="/img/docs/nom-de-domaine/code-transfert/obtenir-code-autorisation-1and1-4.png"
    title="Débloquer et récupérer le code de transfert d'un domaine chez Ionos"
    caption="Récupération du code de transfert chez Ionos"
    alt="Capture d'écran de la procédure de récupération du code de transfert ionos"
/>

Vous pouvez renseigner ce code de transfert dans votre espace client o2switch pour initier le transfert de domaine.
